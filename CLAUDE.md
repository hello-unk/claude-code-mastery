# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

개발자 웹 이력서 프로젝트 - HTML, CSS, JavaScript, TailwindCSS를 활용한 반응형 웹 이력서

## 언어 및 커뮤니케이션 규칙

- **기본 응답 언어**: 한국어
- **코드 주석**: 한국어로 작성
- **커밋 메시지**: 한국어로 작성
- **문서화**: 한국어로 작성
- **변수명/함수명**: 영어 (코드 표준 준수)

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- TailwindCSS (CDN 또는 npm)

## 프로젝트 구조

```
/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # JavaScript 기능
└── assets/
    └── images/         # 이미지 리소스
```

## 개발 명령어

```bash
# TailwindCSS CDN 사용 시 - 로컬 서버 실행
npx serve .

# 또는 Live Server (VS Code 확장) 사용
```
